# Jovi Smartphone - Sprint 2
 Consiste numa interface de login e cadastro para um sistema de smartphones, utilizando apenas **JAVA SCRIPT***

## 🚀 Funcionalidades
- **Login:** Validação de credenciais via JavaScript.
- **Cadastro:** Sistema de registo com redirecionamento direto para a área de boas-vindas.
- **Área do Utilizador (Dashboard):** Página de boas-vindas após autenticação ou registo.
- **Slideshow Dinâmico:** Carrossel de ícones sociais (Google, Instagram, Email) manipulado via DOM.
- **Segurança Visual:** Funcionalidade de mostrar/esconder senha.
- **Recuperação de Acesso:** Uso de `prompt` e `alert` para simulação de recuperação de senha.
